# EmoteResizer

Free online tool to resize Twitch emotes instantly.

**Website:** [emoteresizer.net](https://emoteresizer.net)

## Features

- Resize images to exact Twitch emote sizes (28x28, 56x56, 112x112)
- Support PNG, GIF, JPEG formats
- 100% free, no signup required
- Fast and works entirely in your browser
